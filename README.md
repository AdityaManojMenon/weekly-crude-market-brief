# WTI Crude Market Brief

**Weekly EIA-timed commodity research | Initiated March 2026**

---

## Overview

This repository is a live, weekly publication record of WTI crude oil market analysis — structured around EIA data releases and grounded in quantitative supply-demand signals. Every Thursday following the Wednesday EIA petroleum status report, a new brief is generated, interpreted, and committed here.

This is not a dashboard. It is an analyst workflow — a systematic, repeatable process for translating raw inventory data into documented directional views on WTI.

---

## What This Produces

Each weekly brief covers three analytical layers:

**Inventory & Surprise**
EIA crude and Cushing stock changes measured against the 5-year seasonal average and analyst consensus. The surprise — actual draw/build minus expected — is the primary directional input. A larger-than-expected draw in a structurally tight market reads differently than the same number during a demand slowdown.

**Refinery Utilization**
Utilization rate trends contextualize demand-side pull on crude. Rising utilization with a draw confirms demand-driven tightness. Rising utilization with a build signals upstream supply overwhelming downstream absorption — a divergence worth flagging.

**WTI Futures Curve Structure**
M1–M2 spread and the full 1–12 month curve shape are classified as backwardation or contango. Curve structure is treated as the market's aggregate signal on physical tightness — one of the most honest real-time reads available. Roll yield direction is noted where relevant to positioning context.

---

## Directional Call Framework

Each brief closes with a documented WTI directional call — bullish, bearish, or neutral — with a conviction level and stated risk to view.

The call is generated by cross-referencing inventory signal against curve signal:

| Inventory | Curve | Call | Conviction |
|---|---|---|---|
| Draw > consensus | Backwardation | Bullish | High |
| Build > consensus | Contango | Bearish | High |
| Draw > consensus | Contango | Cautious bullish | Moderate |
| Build > consensus | Backwardation | Cautious bearish | Moderate |
| In-line | Either | Neutral | Low |

Divergence cases — where the physical and the paper market disagree — are the most analytically interesting and receive explicit treatment in the interpretation section.

## Pipeline

The pipeline runs in sequence each Thursday after the 10:30 AM ET EIA release:

```
eia_ingestion.py → surprise_model.py → curve_analytics.py → crack_spreads.py → generate_brief.py
```

GitHub Actions automates the full run and commits the new brief to `briefs/`. Each step writes intermediate outputs to `data/eia_cache/` so any stage can be re-run independently without re-fetching upstream data.

## Trade Logic

Signals are generated using a hierarchy:

1. Curve structure (primary signal — forward expectations)
2. Inventory surprise (secondary — backward-looking)
3. Divergence detection (highest priority)

In divergence regimes, curve structure is treated as dominant, reflecting physical market conditions over headline data.
